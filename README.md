A full-stack Job Portal System built with microservices architecture. Backend REST APIs developed with Spring Boot (Java) for user management, job listings, and application tracking. Frontend built with React + TypeScript served via Nginx. Each service is containerized with Docker using multi-stage builds, managed across Development, Testing, and Production environments with Docker Compose. Services are orchestrated and scaled using Kubernetes with Deployments, StatefulSets, ConfigMaps, Secrets, PersistentVolumes, and Ingress. Monitoring and logging implemented with Prometheus scraping Spring Boot Actuator metrics and visualized through Grafana dashboards. Deployed on Linux (Ubuntu) with PostgreSQL as the database for each service.
